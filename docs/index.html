An atlas of worlds grown from single seeds. Terrain, seas, climate, rivers, biomes, provinces, cities, roads, and a written history — each layer derived, in order, by a deterministic engine with zero dependencies. Same seed, & same world, forever. Every plate has a labeled poster and a gazetteer; regenerate any with node src/cli.ts generate --seed <name>.
